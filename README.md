# E-Charging Dashboard

Beheerportaal voor laadinfrastructuur, sessies en financiële afrekeningen.

## Stack

- Vite + React 18 + TypeScript
- Tailwind + shadcn/ui
- Supabase (auth + database)
- React Query, React Router

## Development

```bash
npm install
npm run dev
```

App draait op http://localhost:8080.

## Scripts

- `npm run dev` — dev server met HMR
- `npm run build` — productiebuild
- `npm run preview` — preview van productiebuild
- `npm run lint` — eslint
- `npm run test` — vitest
